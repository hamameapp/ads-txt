Privacy Policy / プライバシーポリシー

This Privacy Policy explains how ナンプレ先生 - スキャン＆解説付き数独 handles user data.

Last updated: 2026-03-13

1. App covered by this policy / 対象アプリ

This policy applies to the Android application ナンプレ先生 - スキャン＆解説付き数独.

本ポリシーは、Androidアプリ ナンプレ先生 - スキャン＆解説付き数独 に適用されます。

2. Data the app accesses or uses / アプリがアクセスまたは利用する情報

(a) Camera / カメラ

The app requests camera permission only when the user chooses the OCR feature to capture a Sudoku board. The camera is used to take a photo of the puzzle image for board recognition.

本アプリは、ユーザーがOCR機能で数独盤面を撮影する場合にのみ、カメラ権限を要求します。カメラは盤面認識のための撮影に使用されます。

(b) Photos selected by the user / ユーザーが選択した画像

If the user chooses an image from the device, that image is used only for OCR processing and board import.

ユーザーが端末内の画像を選択した場合、その画像はOCR処理および盤面読み込みのためにのみ使用されます。

(c) Puzzle data saved by the user / ユーザーが保存したパズル情報

If the user uses the stock/save feature, the app stores puzzle-related data such as board digits, titles, and creation time on the device.

ユーザーがストック保存機能を利用した場合、盤面の数字、タイトル、作成日時などのパズル関連データを端末内に保存します。

(d) Advertising-related data / 広告関連データ

The app uses Google AdMob to serve banner and interstitial ads. Through the Google Mobile Ads SDK, Google may automatically collect data such as IP address, advertising identifiers, app interaction data, and diagnostic information for ad delivery, measurement, fraud prevention, and service improvement.

本アプリは、バナー広告およびインタースティシャル広告の配信に Google AdMob を使用します。Google Mobile Ads SDK を通じて、Google は広告配信、測定、不正防止、サービス改善のために、IPアドレス、広告IDなどの識別子、アプリ操作情報、診断情報等を自動的に収集する場合があります。

3. How the data is used / 利用目的

  • To capture or import Sudoku board images for OCR recognition.
  • To process the selected or captured image on the device and convert it into puzzle data.
  • To save puzzle data locally when the user uses the stock/save feature.
  • To display ads and support ad measurement and fraud prevention through AdMob.
  • 数独盤面画像の撮影または読み込みを行い、OCR認識に利用するため
  • 選択または撮影した画像を端末内で処理し、盤面データに変換するため
  • ユーザーがストック保存機能を利用した際に、パズルデータを端末内へ保存するため
  • AdMob を通じて広告を表示し、広告配信・測定・不正防止を行うため

4. Data sharing / 第三者提供

The developer does not intentionally send captured or selected Sudoku images to developer-managed servers for OCR processing in this version of the app. Puzzle data saved through the stock feature is stored locally on the user’s device.

このバージョンのアプリでは、開発者は、撮影または選択された数独画像をOCR処理のために開発者管理サーバーへ意図的に送信しません。ストック機能で保存されたパズルデータは、ユーザー端末内に保存されます。

However, when ads are displayed, data may be handled by Google and its partners in accordance with their own policies.

ただし、広告表示時には、Google およびそのパートナーにより、各社のポリシーに従ってデータが取り扱われる場合があります。

5. Data storage, retention, and deletion / 保存期間・削除

Images captured for OCR may be temporarily created in the app cache on the device. Saved puzzle data remains on the device until the user deletes it from within the app or uninstalls the app.

OCR用に撮影した画像は、端末内のアプリキャッシュに一時的に作成される場合があります。ストック保存されたパズルデータは、ユーザーがアプリ内で削除するか、アプリをアンインストールするまで端末内に保持されます。

6. Security / 安全管理

The app is designed to minimize data access and to use only the permissions necessary for the features offered. Data handled by third-party SDKs is subject to the security practices of those providers.

本アプリは、提供機能に必要な範囲でのみ権限を利用し、データアクセスを最小限にするよう設計されています。第三者SDKによって取り扱われるデータは、当該提供者のセキュリティ方針に従います。

7. Third-party services / 外部サービス

  • Google AdMob / Google Mobile Ads SDK
  • Google Play services
  • Google ML Kit (text recognition used for OCR processing)

8. Children’s privacy / 子どものプライバシー

The app is a Sudoku utility/game application and is not intended to collect personal information from children knowingly.

本アプリは数独支援・ゲーム用アプリであり、子どもから個人情報を意図的に収集することを目的としていません。

9. Changes to this policy / 改定

This policy may be updated when app features, SDKs, or legal requirements change. The latest version will be posted on this page.

本ポリシーは、アプリ機能、SDK、法令等の変更に応じて改定される場合があります。最新の内容は本ページに掲載します。
